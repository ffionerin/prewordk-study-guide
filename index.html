HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • You open an element using these brackets and close them using the brackets plus a forward slash
  • Elements include unordered lists (ul) ordered lists (ol) and paragraphs (p)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • I can used code such as "color:red", "width: 100%", "height:130px" to change the style of the webpage
  • I need to remember to link the CSS code to the HTML code by including it in within the "head" section

Git

  • git status: checks what brack we are currently on
  • git checkout -b branch name: cretes a new branck and switches to it
  • git commit: takes the staged snapshot and commits it to the project history (Combined with git add, this defines the basic workflow)
  • git pull: Downloads a branch from a remote repository and immediately merges into the current branch
  • git push: let's you move a local branch to another repository

Javascript

  • Whereas the CSS code is linked to HTML within the 'head' element, JavaScript is linked to HTMl within the body element
  • JavaScript allows users to interact with the application
  • It allows us to set up if statements and loops
  • 'For' lopps are made up of the keyworkd 'for', then a three-part conditional statement and then a code block
